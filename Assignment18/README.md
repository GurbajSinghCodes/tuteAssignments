#  JavaScript DOM and Event Handling
- This assignment illustrates use of event handlers to dynamically update :
 - the greet message using value from input field upon button click
 - the background color of boxes matching to the color written inside their text field

## How to use:
- Open [index.html ](index.html) directly in your browser
- Enter your name in the input field and either press **Enter** or click the **Greet** button, the greeting message on top updates after that
- Click in any of the four boxes to change their background color to the color name written inside them
