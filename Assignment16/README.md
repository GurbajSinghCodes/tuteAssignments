# Map filter exercise
- This assignment demonstartes how to dynamically populate cards using javascript and apply search on them using filter function
- The **Filter** function filters required data from the given list
- This assignment showcases the use of event listerners to trigger searches
## How to run
- Open the [index.html](index.html) file 
- The website loads with multiple cards at landing page having student details
- Search using the searchbar to filter out the required results
