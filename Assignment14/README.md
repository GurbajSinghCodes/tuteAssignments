# Assigment to check various properties of a given variable 
## ⚡This program performs the following functions
- Returns the sum of first n numbers where n is the given variable
- Prints the table of given variable
- Checkc whether the given number is prime or not prime 
- Return the factors of the given number    
- Gives the sum of digits of the given number 
- Checks id the given number is an Armstrong number

## ⚡Usage
- This assignment showcases the use of loops and conditional statments 
- COnverts the given number to String and back to Number at various positions for purpose of loop and applying different operations on digits of number
