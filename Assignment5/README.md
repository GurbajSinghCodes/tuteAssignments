# Laundry Services Hero Section

---

## 📌 Overview
- This assignment has a landing page for a laundry service including a **Navbar** and **Hero Section**
- Sections need to be placed without using **"Flex"** so positioning properties have been used 

---

## 🏗 Basic structure
- A Navbar at the top having a logo, navigation sections and Username
- A **Hero Section** divided into two parts as left and right
- Left one has the basic info about laundry services, right one has an image depicting Laundry services

---

## 🎨 CSS
- All styling is done using a single external CSS file **[style.css](./style.css)**
- Styling starts with universal styles using * and body
- There are section specific styles under each section's comment
