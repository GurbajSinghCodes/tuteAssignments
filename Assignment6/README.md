# RinseUp Laundry Service – Login Page

Login page UI for a laundry service platform called **RinseUp**.
The interface provides users with a simple way to sign in and access laundry service features.

---

## 📌 Overview

RinseUp is a web interface designed for an online laundry service.
This page allows existing users to log in using their **email and password** while also providing options for **password recovery** and **new user registration**.

The design focuses on:

* Clean layout
* Simple user experience
* Friendly visuals related to laundry services

---

## 🖼️ UI Preview

The login page consists of two main sections:

### Left Section

* Brand logo **RinseUp**
* Illustration showing laundry service interaction
* Visual theme with bubbles and laundry environment

### Right Section

* Login form
* Email input field
* Password input field
* Forgot password link
* Login button
* Registration option for new users

---

* **HTML5**
