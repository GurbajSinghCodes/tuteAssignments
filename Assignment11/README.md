# CSS Mini Project - Thala 7 Sneaker Store
## Basics
- 
- This CSS mini project represents a shoe-store named Thala (7 by MS Dhoni)
- It hightlights the use of various CSS properties used throughtout the project development
- There are multiple sections inside each webpage in the project 
- A navbar always appears on every page to help users navigate easily to the pages
- Properties like color, margin, padding, background-color, sizing etc. are customised for each section
- Use of hover effects at various points 

## Guide to Use
- Run the [index.html](index.html) file to get to the landin page of the project 
- Clicking on Shop Now button takes users to the product display section where customers can see product price and various images arranged in a queue
- Clicking the cart icon at any instand of time takes to the cart section to add or remove products or to directly checkout the added items.
- The pay now button in cart section redirects to the payment page where users can choose from a number of options to pay like Credit card, Bank transfer, Bitcoin payments etc.
